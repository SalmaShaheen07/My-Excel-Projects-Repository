# Data Technician Bootcamp – Excel Project Summary

This repository contains my completed Excel project from the Data Technician Bootcamp (May–June 2026).
Across several practical labs, I worked with retail sales data, county‑level product sales, and bike sales datasets to build confidence in Excel for real‑world data analysis.

The project demonstrates core data‑handling, cleaning, visualisation and analytical skills essential for entry‑level data roles.

## Project Overview
Throughout the bootcamp, I completed a series of structured Excel tasks involving:

Retail sales analysis

County‑level product performance

Bike sales pivot tables and visualisations

Formula‑based categorisation and summarisation

Data cleaning, sorting, and filtering

PivotTable creation and interpretation

Chart building for insights and storytelling

This README summarises the skills and techniques applied.

## Key Excel Skills Demonstrated
### Formulas & Functions
I used a wide range of Excel formulas to calculate, summarise, and categorise data:

SUM / SUMIF – calculating totals and conditional totals

AVERAGE / AVERAGEIF – computing overall and conditional averages

DATE, MONTH, YEAR – extracting and manipulating date components

UNIQUE – generating distinct lists for validation and analysis

VLOOKUP – performing lookups across datasets

SWITCH – categorising sales volumes into High, Medium, and Low

Logical formulas using TRUE, >=, > and nested conditions

Example SWITCH formula used:

Code
=SWITCH(TRUE, C2 > 600, "High", C2 >= 300, "Medium", "Low")

### Data Cleaning & Preparation
